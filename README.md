# FITNESS-TRACKER

Мини-система для сбора и анализа данных фитнес-тренировок.

- **Генератор данных (Python)** — каждые 2–5 секунд создаёт тренировки для случайного пользователя.
- **PostgreSQL** — хранит пользователей, тренировки, статистику.
- **Redash** — дашборды и визуализации.
- **Redis** — используется Redash для кэширования.
